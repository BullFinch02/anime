# 🎌 Anime VF France

Catalogue d'animés doublés en français — hébergé sur GitHub Pages.

## 🚀 Déploiement sur GitHub Pages

### Étape 1 — Créer le dépôt

1. Va sur [github.com](https://github.com) et connecte-toi
2. Clique sur **New repository** (bouton vert en haut à droite)
3. Nom du dépôt : `anime-vf-fr` *(ou ce que tu veux)*
4. Coche **Public**
5. Clique **Create repository**

### Étape 2 — Uploader les fichiers

1. Dans ton nouveau dépôt, clique **uploading an existing file**
2. Glisse-dépose **tous les fichiers** de ce dossier :
   - `index.html`
   - `README.md`
3. Clique **Commit changes**

### Étape 3 — Activer GitHub Pages

1. Dans ton dépôt, clique sur **Settings** (onglet en haut)
2. Dans le menu gauche, clique **Pages**
3. Sous *Branch*, sélectionne **main** puis **/ (root)**
4. Clique **Save**
5. Attends 1-2 minutes ⏳

### Étape 4 — Accéder au site

Ton site sera disponible à l'adresse :
```
https://TON_PSEUDO_GITHUB.github.io/anime-vf-fr/
```

---

## 🔑 Clé API Claude (pour les traductions)

Le site utilise l'API Claude pour traduire les titres et synopsis en français.

1. Va sur [console.anthropic.com](https://console.anthropic.com)
2. Clique **API Keys** → **Create Key**
3. Copie la clé
4. Dans le site, clique sur ⚙️ **Paramètres** et colle ta clé

> La clé est sauvegardée uniquement dans ton navigateur (localStorage), elle n'est jamais envoyée ailleurs.

---

## 📦 Mise à jour du site

Pour mettre à jour le site après une modification :
1. Va dans ton dépôt GitHub
2. Clique sur `index.html`
3. Clique l'icône ✏️ (crayon) en haut à droite
4. Colle le nouveau contenu
5. Clique **Commit changes**

Le site se met à jour automatiquement en ~1 minute.
